# Space-Invaders

Space Invaders done in Java-17, played entirely through the console using the 'WASD' keys.

## Controls

Controls in menu:  
`w` : move up  
`s` : move down  
`d` : select  

Controls in game:  
`a` : move left  
`d` : move right  
`s` : shoot
